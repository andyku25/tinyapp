# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly). This app currently utilizes an object as a database. Implementation of a database my be included in a future version. Features indcluded in the below screenshot are as described:
- Registration Form
- Login Form
- Create new short URL
- Read your short URL links 
- Update existing links
- Delete your links

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
